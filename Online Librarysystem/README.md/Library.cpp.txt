# Online Library Management System (C++)

## Description
This project is a C++ implementation of an Online Library Management System using Object-Oriented Programming principles.

## Features
- Add and remove books
- Search books by title
- Manage users
- Borrow and return books
- Includes positive and negative test cases

## OOP Concepts Used
- Encapsulation
- Abstraction
- Modularity

## How to Run
1. Compile: g++ library.cpp -o library
2. Run: ./library
