# Online Library Management System (C++)

This project is a C++ implementation of an Online Library Management System using Object-Oriented Programming (OOP) principles. 
It allows management of books and users, supports borrowing and returning of books, and includes test cases to validate system functionality.

## Features
- Add and remove books
- Add and manage users
- Search for books by title
- Borrow and return books
- Includes positive and negative test cases

## OOP Concepts Used
- Encapsulation
- Abstraction
- Modularity
- Reusability & Maintainability

## How to Run
1. Open library.cpp in Dev C++.
2. Compile: Execute → Compile (F9)
3. Run: Execute → Run (Ctrl + F10)
4. The console will show book management operations.
